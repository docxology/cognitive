# Attention Networks

Attention networks are the distributed neural systems that implement attentional control and processing in the brain. From an active inference perspective, these networks orchestrate precision-weighting of prediction errors and coordinate information flow to minimize free energy across multiple processing levels.

## Core Networks

### Alerting Network
1. **Components**
   - Locus coeruleus
   - Thalamic nuclei
   - Frontal regions
   - Parietal areas
   - Precision modulators

2. **Functions**
   - Arousal maintenance
   - Vigilance control
   - Response preparation
   - State regulation
   - Uncertainty management

### Orienting Network
1. **Components**
   - Superior parietal lobule
   - Temporal parietal junction
   - Frontal eye fields
   - Superior colliculus
   - Precision controllers

2. **Functions**
   - Spatial attention
   - Feature selection
   - Location prioritization
   - Search guidance
   - Prediction updating

### Executive Network
1. **Components**
   - Anterior cingulate
   - Lateral prefrontal
   - Basal ganglia
   - Anterior insula
   - Integration hubs

2. **Functions**
   - Control implementation
   - Conflict resolution
   - Goal maintenance
   - Performance monitoring
   - Model optimization
